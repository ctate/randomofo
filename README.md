# randomofo 

randomizer for veganmofo.com

# Install randomofo

1. Install [Node.js](https://nodejs.com)
2. Install Crystal CLI: `npm install crystal-cli -g`
3. Clone randomofo: `git clone https://github.com/ctate/randomofo`

# Build randomofo

```sh
cd /path/to/ctate/randomofo
crystal build
```

# Run randomofo

```sh
cd /path/to/ctate/randomofo
crystal run
```
